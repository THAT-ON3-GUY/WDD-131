My Interests Site Plan

Jayden Pope

WDD 131-A1

Overview

Purpose

The purpose of the site is to further my web design experience through making a page that informs those that visit it about some of my personal interests.

Audience

Those that want to know more about me.

Dynamic elements

A button that displays a random fun fact from an array for my home page. Create an array of objects for my child page that contains info on games I like. Display games I like dynamically using .map() and DOM interaction Have a search bar that let's the guest filter and search to see if certain games are on my list. Dropdown genre filter that helps filter searches. Counter at top of games page that show the number of games that fit the active criteria.
